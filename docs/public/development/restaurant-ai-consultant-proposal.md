
**普通の言葉で質問するだけで、AIがデータに基づいた分析と提案をしてくれます。**

---

## 📊 AIが見てくれるデータ

お店に関するあらゆる情報をAIが記憶して、分析に活用します。

| 分類 | 具体的なデータ |
|------|---------------|
| **💰 売上** | 日々の売上、客単価、時間帯別の売上など |
| **🍽️ メニュー** | どのメニューが売れているか、利益率はどうか |
| **⭐ 口コミ** | Googleマップ、食べログ、Instagramの評価 |
| **👥 スタッフ** | シフト情報、人件費 |
| **📦 仕入れ** | 食材の仕入れ、在庫、廃棄ロス |
| **🌤️ 外部情報** | 天気、近くのイベント、競合店の動き |
| **📝 あなたのメモ** | 日々の気づき、お客様の声、アイデア |

## 🗣️ AIに相談できること

### 売上について
- 「今月の売上はどう？」
- 「なんで水曜日は売上が低いの？」
- 「利益を増やすにはどうしたらいい？」

### メニューについて
- 「どのメニューが人気？どれが不人気？」
- 「新メニューのアイデアある？」
- 「この料理、値上げしても大丈夫？」

### お客様の評判について
- 「最近の口コミはどんな感じ？」
- 「クレームで多いのは何？」
- 「リピーターを増やすにはどうしたらいい？」

### スタッフ・運営について
- 「明日のシフト、人足りてる？」
- 「人件費を抑えるにはどうしたらいい？」

### 集客・マーケティングについて
- 「SNSで何を発信したらいい？」
- 「どんなキャンペーンが効果的？」
- 「競合店との違いは何？」

### 予測について
- 「来週の来客数はどのくらい？」
- 「注意しておくべきリスクはある？」
